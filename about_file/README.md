###读写文件
####Python操作文件的相关写法
- os.path.join()函数很有用，可以用来处理不同操作系统的不同分隔符
- os.getcwd()获取当前的工作目录，os.chdir()可以改变工作目录
- 绝对路径和相对路径，如果我们写程序时使用到文件(比如配置文件)，那我们应该将配置文件的路径设置为相对路径。绝对路径：总是从根文件夹开始；相对路径：相对于程序的当前路径。
- os.makedirs()创建新文件夹
- os.path模块包含了许多与文件名和文件路径相关的有用函数。
