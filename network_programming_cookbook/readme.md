### 该选用TCP还是UDP的一些指导
#### 您应该使用TCP，如果：
- 您需要一个可靠的数据传输，以确保您的数据完整无缺地到达目的地；
- 您的协议需要不止一个请求和服务器的回答；
- 您要发送较多的数据；
- 初始连接出现的短暂的延迟是可以容忍的。

#### 您应该使用UDP，如果：
- 您不太关心信息包是否到达或者不太在意信息包到达的殊勋是否正确，再或者您可以自己察觉这些问题并自己解决；
- 您的协议只包含基本请求和回答；
- 您需要尽快建立网络会话；
- 只传送很少一部分数据。Udp的限制是一个数据包不超过64KB的数据，通常人们只用UDP传送1KB以下的数据。
